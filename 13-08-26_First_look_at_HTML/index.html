!DOCTYPES

Doctypes appear at the very top of the page. They tell the browser how the HTML on your page should be interpreted.

XHTML 1.1

XHTML 1.1 is a form of xml and the strictest doctype. Forcing xml rules on the document. The browser doesn't care though.

<?xml version="1.0" encoding="utf-8"?>
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">
<html xmlns="http://www.w3.org/1999/xhtml">

HTML5

The HTML5 doctype allowes code to be written far more loosely.

<!DOCTYPE html>
<html>

BLOCK elements

Block elements provide the main structure of the page. They consume the entire width available to them.

Headings

Divisions

The div element is used for parts of the document that can't be semantically described by other html block elements. It is usually used in combination with a class or id attribute.

<div class="article">
   Some very exciting news!
</div>

Paragraphs

The paragraph element encloses paragraphs.

<p>
   This is a paragraph.
</p>

INLINE elements

An inline element does not break the flow of the page.

Images

Images can be included in your page with this tag <img src="img/devok.jpg />.

Span

The span element is the inline equivalent of a div. It can be used to mark up elements that can not be described by any other element.

Example: I read The art of computer programming.

Code: I read <span class="book">The art of computer programming</span>.

Strong

The strong element is used for lifting out text that is important.

<strong>Important!</strong>

Emphasis

The Emphasis element is used for lifting out text that is important but not as important as strong text.

<em>Important!</em>
